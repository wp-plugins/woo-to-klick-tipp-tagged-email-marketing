WooCommerce plugin (free)<br>(English description below German version!)<br><h4>DE:</h4><strong>Basic Plugin Version (kostenlos)</strong><ul><li>Synchronisiert Kunden und Bestell-Status<li>Keine Synchronisation wenn der Bestell-Status sich ändert z.B. abgebrochen, zurückerstattet<li>Keine Synchronisation der gekauften Produkte/Kategorien etc.</ul><blockquote>Premium Plugin Version:Synchronisiert und erstellt folgende Tags:<ul><li>Bestellstatus<li>Produkte die gekauft wurden (bei Bestellungen mit dem Status "completed"/abgeschlossen)<li>Produktkategorien<li>Umsatz je Bestellung in 50 € Schritten<li>Eingekauft in den letzten 6 Monaten</ul>Synchronisierte Daten über die Extra Felder: Kundenname, Firma, Adresse, TelefonSynchronisierte Daten über die Extra Felder (der letzten Bestellung): Bestellnummer, Bestellstatus, exakter Umsatz, Produkte, Versandkosten, RabatteTipp: Wenn Sie in der Klick-Tipp Kontakt Cloud nach Extra Felder filtern, können Sie dem Ergebnis manuell tags zuweisen</blockquote>Synchronisierungsablauf am Beispiel: Kauft der Kunde in Ihrem Shop wird er automatisch zu Klick-Tipp synchronisiert inkl. double opt-in.Je nach Zahlungsart bekommt er den Bestellstatus, "Wartend", "In Bearbeitung", ...<br>Weitere Tags werden dem Benutzer hinzugefügt wenn sich der Bestellstatus auf "Abgeschlossen" ändert.Z.B. Produkte die gekauft wurden, Produktkategorien, Umsatz je Bestellung in 50 € Schritten, Eingekauft in den letzten 6 Monaten.Wird eine Bestellung gutgeschrieben/abgebrochen wird auch der Bestellstatus sich ändern und Tags ggf. entzogen.Technik: WP-Cron wird zum auslösen der Synchronisierung verwendet.Die Synchronisierung erfolgt über die REST API Schnittstelle. REST API wird sowohl von WooCommerce als auch von Klick-Tipp von Haus aus unterstützen.<br><blockquote><h4>Unterstützung:</h4> Das Plugin wurde von einem der ersten Klick-Tipp Certified Consultants Tobias Conrad
