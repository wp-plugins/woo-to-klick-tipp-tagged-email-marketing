tag based email marketing for woocommerce</strong>This plugin requires Klick-Tipp Account (paid see FAQ) and the WooCommerce plugin (free).<strong>Basic Plugin Version (Free)</strong>Sync customer and order StatusNo sync when order Status is canceled or refunded ordersNot sync products bought<strong>Premium Plugin Version</strong>sync and create tags: order status, product(s) bought (on status "completed"), order amount in 50 digi steps, bought whihin last 6 monthsynced data in extra fields: Customer name, company, adress, telephonesynced data in extra fields only last order: order number, order status, Total Amount, products, shipping fee, discount<strong>Tip</strong>: When you filter Contact Cloud by extra fields, filtered customers could set with tag manually<strong>Order of events</strong>: When a customer orders at your shop he is auto synced to Klick-tipp.com and get order status tag "on-hold", "progressing" ...Additional tags will be added when order status is "completed". e.g. "product", "order amount" more see above "sync and create tags"If the order is refunded/canceled tags will change also.<strong>Technical</strong>: Trigger sync via external cron, wp-cron, manual or do a complete resync of ordersSync is processed via REST API, natively supported by WooCommerce and Klick-TippSupport: We will support you in time and with a good mind.Get Pro License on <a href="http://www.woocommerce-klick-tipp.com" target="_blank" rel="nofollow">www.woocommerce-klick-tipp.com</a>== Installation ==1.
